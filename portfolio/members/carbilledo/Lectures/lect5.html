Module 4: Images

Inserting images with <img>

    In HTML, images are included using the tag. This tag does not have a closing tag; instead,
    it's self-closing.

    Here's how you include an image on a webpage:

    <img src="image_url_here" alt="alternative_text_here">


    Explanation of attributes:

    • 'src' : Specifies the URL or path to the image file. This can be a relative or absolute URL.
    • 'alt' : Specifies alternative text for the image. This is important for accessibility purposes and
      should describe the content of the image in case it cannot be displayed.

    Example:

    <img src="example.jpg" alt="Example Image">

    In this example, example.jpg is the image file, and "Example Image" is the alternative text.


    Additionally, you can include images from other websites by providing their URLs in the 'src' attribute. These are often referred to as "image links."

    For example:


    <img src="https://ue.instructure.com/login/canvas" alt="Canvas Instructure">

    This would display the image hosted on 'https://ue.instructure.com/login/canvas'.
