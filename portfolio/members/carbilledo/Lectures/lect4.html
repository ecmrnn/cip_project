Module 3: Links

Creating Links with <a>

    Links, in the context of web pages, are clickable elements that allow users to navigate
    between different pages or sections of a website, or to external resources on the internet.
    They are fundamental to web navigation and play a crucial role in connecting content
    together.



    To create links on a webpage in HTML, you use the (anchor) tag. Here's the basic syntax:

    <a href="URL"> Link Text </a >



    Here's an examples:

    <a href="https://www.example.com"> Example Link </a >

    This will create a link that says "Visit Example Website" and, when clicked, will take the user to
    the website https://www.example.com.


    Additionally, you can create internal links within the same webpage or to different sections
    of the same webpage using the 'id' attribute and the '#' symbol followed by the ID of the target
    element.

    For example:

    <a href="#section2"> Go to Section 2 </a >

    This link would take the user to the section of the webpage with the ID "section2".
